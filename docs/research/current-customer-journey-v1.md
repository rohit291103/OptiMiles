---

# Current journey overview

```
Travel Dream → Google Search → YouTube videos → Blogs & Reddit
→ Compare cards → Calculate rewards manually → Ask communities
→ Make a guess → Hope it works
```

Largely manual, and lacking confidence at every step.

---

# Stages, pains, and opportunities

| Stage | What the user does | Pain points | OptiMiles opportunity |
|---|---|---|---|
| **1. Travel inspiration** | Sees premium travel on Instagram/YouTube; reads blogs | No idea whether the goal is realistic, how many points, where to start | Replace inspiration with a concrete plan |
| **2. Information search** | Googles "best travel credit cards India", KrisFlyer, Bonvoy | Conflicting recommendations, outdated articles, sponsored reviews, overload | Trustworthy, centralized recommendations |
| **3. Card evaluation** | Multiple tabs, comparison tables, spreadsheets, videos | Ecosystems hard to compare; nothing personalized; rules complex | Goal-based recommendations, not generic rankings |
| **4. Reward planning** | Manual estimates of required points, earning rate, time-to-goal | Error-prone, slow, can't account for milestones/caps/ratios | Automated calculations and timeline projections |
| **5. Decision** | Apply / keep existing / delay / abandon | Fear of the wrong decision; fee uncertainty; no confidence | Explainable, data-backed strategies |
| **6. Execution** | Spends, tries to remember categories, checks balances occasionally | Forgets which card; misses milestones; suboptimal allocation | Future: progress tracking, reminders, spend guidance |
| **7. Redemption** | Attempts transfer + booking | Unsure when to transfer, ratios opaque, enough points?, rules confusing | Transfer path + redemption readiness visualization |

# Emotional journey

Inspiration: excitement → Search: curiosity → Comparison: **confusion** → Planning: **overwhelm** → Decision: **anxiety** → Execution: uncertainty → Redemption: satisfaction (if successful) or frustration.

A successful product reduces confusion and anxiety while increasing confidence.

# Existing alternatives

Google, YouTube creators, Reddit communities, travel blogs, comparison websites, bank reward pages, personal spreadsheets, friends' advice. **No single solution provides end-to-end strategy planning.**

# Key pain points (summary)

Fragmented information · ecosystems hard to understand · generic recommendations · manual calculations · low decision confidence · tools focus on cards rather than goals.

# Product opportunity

Replace disconnected tools with one goal-driven experience. Instead of "Which credit card is best?", the user asks "How do I achieve my travel goal?" — and the product answers: Is it achievable? How many rewards? Which cards? How to allocate spend? How long? Why this strategy?

# Success definition

The journey is transformed when a user moves from **Travel Dream → Actionable Strategy in under a minute**, confident the recommendation is accurate, explainable, and achievable.
