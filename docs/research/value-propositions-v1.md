# Customer profile

**Target customer:** a travel aspirant seeking premium travel through intelligent reward optimization — moderate-to-high monthly spending, limited reward-ecosystem understanding, willing to optimize if given a clear plan.

**Customer jobs:** determine goal achievability · understand the best strategy · choose the right cards · maximize accumulation · optimize monthly spending · understand transfer partners · build decision confidence.

**Customer pains:** research complexity (blogs, videos, conflicting advice) · decision uncertainty (which card, fee worth it?) · reward complexity (calculations, milestones, caps, ratios, ecosystems) · execution problems (which card, how long, progress?) · trust problems (generic advice, sponsored reviews, black-box AI).

**Desired gains:** a realistic strategy · decision confidence · faster goal achievement · maximum value from everyday spending · simplicity · clear progress tracking · explainable recommendations.

# Value map

**Products & services:** goal-based recommendation engine · reward knowledge engine · spend optimization engine · timeline simulation engine · transfer intelligence · explainable AI · strategy dashboard · progress tracking — one intelligent planning platform.

## Pain relievers

| Pain | OptiMiles solution |
|---|---|
| "I don't know where to start" | Goal-first onboarding — begins with the travel aspiration, not the card |
| "Too much conflicting information" | Centralized recommendations on structured reward intelligence, not sponsored rankings |
| "Reward calculations are confusing" | Automated calculations, milestone tracking, timeline simulations |
| "I don't know which card to choose" | Goal-specific recommendations from existing cards, spend, timeline, preferences, fee tolerance |
| "I don't trust AI recommendations" | Every recommendation ships with supporting calculations, transfer paths, rationale — transparency as a feature |
| "I don't know if my goal is realistic" | Goal achievability analysis before any card recommendation |

## Gain creators

**Confidence** (explainable, evidence-based) · **Simplicity** (ecosystem complexity → one execution plan) · **Faster goal achievement** (every rupee routed efficiently) · **Financial efficiency** (value from spending already planned) · **Visibility** (progress, targets, remaining effort, ETA) · **Education** (users learn ecosystems while using the product).

# Competitive differentiation

Most platforms optimize around cards ("Which card is best? Which gives more cashback?"). OptiMiles optimizes around travel goals ("Can I achieve my goal? What's the smartest strategy? What next? How long?"). Goal-centric planning over product-centric recommendations is the primary differentiator.

# MVP value proposition

Success = a user moves from *"I want to travel using points"* to *"I know exactly how to achieve my goal"* **within a single session**. The MVP provides clarity, confidence, and a trustworthy strategy — not execution automation.

# Value proposition statement

> **OptiMiles is an AI-powered travel reward strategist that helps Indian consumers turn everyday spending into premium travel experiences through personalized, explainable, and goal-based credit card reward optimization.**

Unlike comparison websites or generic AI assistants, OptiMiles begins with the travel aspiration and generates a clear execution plan that maximizes accumulation, optimizes spending, and provides confidence the goal is achievable.

# Final product philosophy

Users don't wake up wanting a new credit card. They wake up wanting experiences — a business-class flight, a luxury stay, a dream vacation. Cards, points, miles and transfer partners are tools; OptiMiles orchestrates them into the smartest path toward the desired experience.
